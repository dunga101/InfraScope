# InfraScope

InfraScope is a lightweight Windows observability agent designed to collect local telemetry and detect system anomalies before they impact users.

## Goals

- Lightweight monitoring agent
- Local anomaly detection
- Structured JSON logs
- Windows-first diagnostics

InfraScope focuses on early signals of instability such as:

- sustained CPU spikes
- memory pressure
- disk issues
- service crashes
- critical Windows event logs
